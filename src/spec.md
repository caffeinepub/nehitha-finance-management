# Specification

## Summary
**Goal:** Remove Aadhar and PAN number fields from the customer request form.

**Planned changes:**
- Remove Aadhar number input field from the form
- Remove PAN number input field from the form
- Update form validation to no longer require these fields
- Update WhatsApp message template to exclude Aadhar and PAN data

**User-visible outcome:** Users can submit customer requests without providing Aadhar or PAN numbers. The form is simpler and WhatsApp messages no longer include these fields.
